# OLIDEV

Site institucional oficial da OLIDEV LTDA.

A OLIDEV é um núcleo de engenharia tecnológica avançada com atuação em software, infraestrutura, automação, observabilidade, cybersecurity e inteligência aplicada para operações institucionais e corporativas.

## Stack

- Next.js
- React
- TypeScript
- TailwindCSS
- Framer Motion

## Execução local

```bash
npm install
npm run dev
